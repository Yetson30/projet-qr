Modules Fonctionnels

Composants Clés

Découpage en 8 modules indépendants pour une maintenabilité maximale.

M1

Éditeur WYSIWYG

Canvas Konva.js avec couches, transformation libre, snap magnétique, alignement, grille.

  • Drag & Drop natif
  • Resize + Rotation libre
  • Snap magnétique ±5px
  • Grille configurable
  • Multi-sélection (Shift+clic)
  • Lock/Unlock calque
  • Undo/Redo (Ctrl+Z/Y, 50 états)
Konva.js 9Transformer API
M2

Templates

Système de templates JSON sérialisés, galerie visuelle, duplication, export/import.

  • Templates A4 (2480×3508 px)
  • Templates A5 (1748×2480 px)
  • Sérialisation JSON Konva
  • Miniatures preview live
  • Templates custom sauvegardables
  • Champs liés ({{nom}}, {{code}})
JSON Schemaelectron-store
M3

Import Données

Ingestion CSV/Excel avec mapping colonnes, validation, prévisualisation avant import.

  • CSV UTF-8 + séparateur auto
  • Excel .xlsx / .xls
  • Mapping colonnes drag & drop
  • Validation regex (code marchand)
  • Préview tableau 10 lignes
  • Ajout individuel formulaire
  • Historique imports
PapaParseSheetJS (xlsx)
M4

Gestion QR Codes

Import dossier PDF/PNG, extraction automatique depuis PDF, recadrage intelligent.

  • Import dossier QR (PDF ou PNG)
  • Extraction PDF → PNG (pdf2pic)
  • Recadrage automatique (sharp)
  • Association nom fichier ↔ code
  • Préview QR inline canvas
  • Fallback QR manquant (placeholder)
pdf2picsharpjimp
M5

Export Print-Ready

Export PDF A4/A5 à 300 DPI fidèle au rendu canvas, PNG haute résolution.

  • PNG 300 DPI (2480×3508)
  • PDF print-ready (pdfkit)
  • PDF multi-pages (batch)
  • Nommage auto (code_marchand)
  • Dossier sortie configurable
  • Barre progression batch
  • Log export JSON
pdfkitsharpcanvas API
M6

Génération Batch

Génération automatisée en masse à partir d'une liste marchands + dossier QR.

  • Queue processing asynchrone
  • Parallélisation (Worker Threads)
  • Progression en temps réel
  • Reprise sur erreur
  • Rapport CSV post-génération
  • 500+ affiches / session
Worker ThreadsNode.js streams
M7

Internationalisation

Interface bilingue FR/EN avec switch dynamique sans redémarrage.

  • Fichiers i18n JSON
  • Switch FR ↔ EN en live
  • Persistance préférence langue
  • Pluralisation
  • Messages d'erreur traduits
i18nextelectron-store
M8

Paramètres & Sécurité

Préférences persistantes, chemins par défaut, validation données, logs d'activité.

  • Dossier export par défaut
  • Thème (clair/sombre)
  • Unités (px/mm/cm)
  • Logs activité (electron-log)
  • Export/Import paramètres
  • Validation code marchand

Canaux IPC — Communication Main ↔ Renderer

contextBridge · preload.js
CanalDirectionParamsRéponseDescription
csv:parse R→M {filePath, delimiter} {rows[], errors[]} Parse CSV/Excel → JSON
qr:extract-folder R→M {folderPath} {qrFiles[], errors[]} Scan dossier + extraction QR PDF
pdf:export-single R→M {canvasDataUrl, merchant, outputPath} {success, filePath} Export PDF A4 300DPI unique
pdf:export-batch R→M {merchants[], templateId, outputDir} stream progress events Génération batch avec progression
batch:progress M→R — {current, total, merchantCode} Mise à jour barre progression
template:save R→M {template} {success, id} Sauvegarde template JSON
file:dialog-open R→M {filters, properties} {filePaths[]} Dialog natif Windows open
file:dialog-save R→M {defaultName, filters} {filePath} Dialog natif Windows save
